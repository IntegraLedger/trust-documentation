# Manage Your Documents

Learn how to manage, search, and organize your registered documents on Integra.

## Your Document Library

Access all your registered documents in the "My Documents" section.

### Document List View

Your documents are displayed with:
- **Thumbnail/Icon** - Visual preview (if available)
- **Title** - Document name
- **Registration Date** - When it was registered
- **Status** - Active, Updated, Archived
- **Category** - Document type
- **Actions** - Quick action buttons

### Sorting and Filtering

Sort documents by:
- Registration date (newest/oldest)
- Title (A-Z, Z-A)
- Category
- Status

Filter documents by:
- Date range
- Category
- Tags
- Status

### Search

Use the search bar to find documents by:
- Title
- Description
- Tags
- Registration ID

## Document Details

Click on any document to view full details:

### Registration Information
- Registration ID
- Blockchain transaction hash
- Registration timestamp
- Last updated

### Document Metadata
- Title and description
- Category and tags
- File type and size
- Document hash

### Blockchain Record
- View on blockchain explorer
- Transaction confirmations
- Block number

## Document Actions

### Share Verification Link {#sharing}

Share proof of your document with others:
1. Click "Share" on the document
2. Copy the verification link
3. Send to anyone who needs to verify the document

Recipients can:
- Upload their copy of the document
- Verify it matches your registered version
- View registration details (if public)

### Download Certificate

Get an official certificate of registration:
1. Click "Download Certificate"
2. Choose format (PDF or image)
3. Save for your records

The certificate includes:
- Document registration details
- QR code for verification
- Blockchain transaction reference
- Official Integra seal

### Update Metadata

You can update certain information:
- Title and description
- Tags and category
- Privacy settings

**Note**: You cannot change the actual document or its hash. Any document changes require a new registration.

### Archive Documents

Archive documents you no longer actively use:
1. Click "Archive" on the document
2. Confirm archival
3. Archived documents are hidden from main view but remain on blockchain

To view archived documents:
- Enable "Show Archived" filter
- Unarchive anytime to restore to main view

## Privacy Settings

Control who can see your document information:

### Public
- Anyone with verification link can see metadata
- Document appears in public verification
- Recommended for certificates and credentials

### Private
- Only you can see full details
- Others can verify but won't see metadata
- Recommended for confidential documents

### Link-Only
- Only those with your verification link can access
- Not discoverable in public search
- Recommended for selective sharing

## Batch Operations

Manage multiple documents at once:
1. Select documents using checkboxes
2. Choose bulk action:
   - Update category
   - Apply tags
   - Archive
   - Download certificates
3. Confirm the operation

## Document History

View the complete history of each document:
- Initial registration
- Metadata updates
- Sharing activity
- Verification attempts

## Export Your Data

Export your document list:
1. Go to Settings
2. Click "Export Data"
3. Choose format (CSV, JSON)
4. Download your complete document registry

## Organization Tips

### Use Categories Consistently
- Create a category system
- Use the same categories across similar documents
- Review and clean up unused categories

### Tag Effectively
- Use multiple relevant tags
- Include dates in tags (e.g., "2024", "Q1-2024")
- Tag by project or client

### Regular Maintenance
- Review documents quarterly
- Archive outdated documents
- Update descriptions for clarity
- Clean up duplicate tags

## Storage and Limits

### Account Limits
- Free tier: 10 documents per month
- Pro tier: 100 documents per month
- Enterprise: Unlimited

### File Size Limits
- Maximum file size: 10MB
- Larger files: Contact enterprise support

### Retention
- Blockchain records are permanent
- Account data retained per privacy policy
- Archived documents remain accessible

## Troubleshooting

### Can't Find a Document
- Check filters and search terms
- Look in archived documents
- Check date range filter

### Sharing Link Not Working
- Link may have privacy restrictions
- Try regenerating the link
- Check recipient has correct file version

### Export Fails
- Try smaller date range
- Check internet connection
- Contact support for large exports
