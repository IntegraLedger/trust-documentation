# Blockchain API

Query blockchain data, transactions, blocks, and document registry information from Integra's blockchain infrastructure.

**Base URL:** `https://api.blockchain.trustwithintegra.com`

## OpenAPI Specification

Download the complete OpenAPI specification to use with your favorite API client:

- [Download OpenAPI Spec (YAML)](/openapi/blockchain-api.yaml)
- [View in Swagger UI](https://petstore.swagger.io/?url=https://78b67064.integra-documentation.pages.dev/openapi/blockchain-api.yaml)

## Quick Start

### Authentication

All requests require an API key in the `Authorization` header:

```bash
curl https://api.blockchain.trustwithintegra.com/v1/chains \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### Available Endpoints

#### Chains
- `GET /v1/chains` - List all supported blockchain networks
- `GET /v1/chains/{chainId}` - Get details for a specific chain

#### Transactions
- `GET /v1/transactions/{hash}` - Get transaction details
- `GET /v1/transactions/{hash}/receipt` - Get transaction receipt
- `GET /v1/transactions/{hash}/logs` - Get transaction event logs

#### Blocks
- `GET /v1/blocks/latest` - Get latest block
- `GET /v1/blocks/{blockNumber}` - Get block by number
- `GET /v1/blocks/{blockHash}` - Get block by hash

#### Documents
- `GET /v1/documents/verify` - Verify document registration
- `GET /v1/documents/{hash}` - Get document details
- `GET /v1/documents/{hash}/history` - Get document history

#### Contracts
- `GET /v1/contracts/{address}` - Get contract information
- `GET /v1/contracts/{address}/events` - Get contract events

#### Addresses
- `GET /v1/addresses/{address}` - Get address transaction history
- `GET /v1/addresses/{address}/balance` - Get address balance

#### Statistics
- `GET /v1/stats` - Get blockchain analytics and metrics

## Code Examples

### Get Chain Information

```typescript
import { fetch } from 'undici';

const response = await fetch('https://api.blockchain.trustwithintegra.com/v1/chains', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
});

const chains = await response.json();
console.log(chains);
```

### Verify Document

```python
import requests

response = requests.get(
    'https://api.blockchain.trustwithintegra.com/v1/documents/verify',
    params={'hash': '0x1234...'},
    headers={'Authorization': 'Bearer YOUR_API_KEY'}
)

document = response.json()
print(document)
```
