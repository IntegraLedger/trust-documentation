# BasicAccessV1Verifier V4.5

Simple access control verifier implementation for document verification.

## Overview

BasicAccessV1Verifier V4.5 provides:
- Basic access verification logic
- Role-based access control
- Permission management
- Integration with verifier registry
- Simple verification workflows

## Key Features

### Basic Verification
Simple implementation of access verification logic.

### Role Management
Role-based access control for different user types.

### Permission Checks

Works seamlessly with IntegraVerifierRegistry.

## Main Functions

### verify

Verify access for a given user and resource.

<Tabs items={['TypeScript', 'Python']}>
<Tabs.Tab>
```typescript
import { ethers } from 'ethers';
import BasicAccessVerifierABI from './abis/BasicAccessV1VerifierV4_5.json';

const contract = new ethers.Contract(
  BASIC_ACCESS_VERIFIER_ADDRESS,
  BasicAccessVerifierABI.abi,
  signer
);

const hasAccess = await contract.verify(
  userAddress,
  resourceId,
  accessLevel
);

console.log('Access granted:', hasAccess);
```
</Tabs.Tab>

<Tabs.Tab>
```python
contract = w3.eth.contract(address=BASIC_ACCESS_VERIFIER_ADDRESS, abi=abi)

has_access = contract.functions.verify(
    user_address,
    resource_id,
    access_level
).call()

print(f'Access granted: {has_access}')
```
</Tabs.Tab>
</Tabs>

## Events

### AccessGranted
```solidity
event AccessGranted(
    address indexed user,
    bytes32 indexed resourceId,
    uint8 accessLevel
);
```

### AccessRevoked
```solidity
event AccessRevoked(
    address indexed user,
    bytes32 indexed resourceId
);
```

## Access Levels

- `0` - No access
- `1` - Read access
- `2` - Write access
- `3` - Admin access
