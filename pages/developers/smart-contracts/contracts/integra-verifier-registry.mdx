# IntegraVerifierRegistry V4.5

Registry for managing authorized verifiers and attestation providers.

## Overview

IntegraVerifierRegistry V4.5 manages:
- Verifier registration and authorization
- Verifier reputation and performance tracking
- Access control for verification operations
- Verifier discovery and lookup
- Attestation provider management

## Contract Interface

<AbiViewer

## Key Features

### Verifier Registration
Register and manage authorized document verifiers.

### Reputation System
Track verifier performance and reputation scores.

### Access Management
Control which verifiers can perform specific operations.

### Discovery
Enable applications to discover and select appropriate verifiers.

## Main Functions

### registerVerifier

Register a new verifier.

<Tabs items={['TypeScript', 'Python']}>
<Tabs.Tab>
```typescript
import { ethers } from 'ethers';
import IntegraVerifierRegistryABI from './abis/IntegraVerifierRegistryV4_5.json';

const contract = new ethers.Contract(
  INTEGRA_VERIFIER_REGISTRY_ADDRESS,
  IntegraVerifierRegistryABI.abi,
  signer
);

const tx = await contract.registerVerifier(
  verifierAddress,
  verifierName,
  verifierMetadata
);

await tx.wait();
```
</Tabs.Tab>

<Tabs.Tab>
```python
contract = w3.eth.contract(address=INTEGRA_VERIFIER_REGISTRY_ADDRESS, abi=abi)

tx_hash = contract.functions.registerVerifier(
    verifier_address,
    verifier_name,
    verifier_metadata
).transact({'from': account.address})

receipt = w3.eth.wait_for_transaction_receipt(tx_hash)
```
</Tabs.Tab>
</Tabs>

### getVerifier

Retrieve verifier information.

<Tabs items={['TypeScript', 'Python']}>
<Tabs.Tab>
```typescript
const verifier = await contract.getVerifier(verifierAddress);

console.log({
  name: verifier.name,
  active: verifier.active,
  reputation: verifier.reputation.toString(),
  verificationsCount: verifier.verificationsCount.toString()
});
```
</Tabs.Tab>

<Tabs.Tab>
```python
verifier = contract.functions.getVerifier(verifier_address).call()

print({
    'name': verifier[0],
    'active': verifier[1],
    'reputation': verifier[2],
    'verifications_count': verifier[3]
})
```
